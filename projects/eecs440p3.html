Camping Simulator

A simulated campground that teaches users to stay safe and be more environmentally conscious.

About the Project

The goal of this project was to create a game that had the potential to make some real social impact. Our inspiration came from all the games out there that claim to be educational. All of them are boring! We wanted to create a game that could be educational, but fun. The key was trying to be engaging first, and then teach the user while they are having fun.

There are a lot of different activities to explore, including: hiking, fishing, roasting marshmallows, and more. There are signs around the campsite that teach the user how to be more environmentally friendly. If you make bad choices, the game will alert you to them. I made basically all of the UI elements for the game, so I'm really proud of them!

This Unreal Engine project was created by myself and four others: Charles Vu, Shawn Halimman, Ashton Thomas, and Justin Lo; for our EECS 440 class at the University of Michigan. Can you believe we made this whole project in just four weeks?

If you want to learn more about the project, there's more information at this project website that our team created, such as dev logs, a project trailer, and weekly progress videos.

Made in Unreal Engine 5

Unreal Engine is one of two major game engines widely used today. Along with Unity, these game engines provide the tools to create an interactable, custom world like the one I made in this project!

Using Blueprints

Blueprints is Unreal Engine's visual coding language. Although code can also be written with C++, the standard language used in Unreal is this node-based language.

And Jira

Jira helped our team keep track of project goals and not fall behind. It is a great tool for measuring progress and documenting work!
