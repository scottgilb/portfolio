Ann Arbor Go!

A Pok√©mon Go-like game where you travel around Ann Arbor to collect tree varieties and fight squirrels.

About the Project

The goal of this project was to create a game that encouraged the players to explore Ann Arbor. The overworld scene with the interactive map was one part of the game, but there was a second scene that used the phone's camera to bring objects into the user's real-world environment using AR (Augmented Reality). Plus, we included a fun zombie-like minigame where you face off against a swarm of squirrels for extra money!

This whole Unreal Engine project was created by myself and one partner, Ashton Thomas, for our EECS 440 class at the University of Michigan. It was really tricky to get some of these objects to spawn and work in two seperate scenes, but the whole thing turned out pretty good!

Made in Unity

Unity is one of two major game engines widely used today. Along with Unreal Engine, these game engines provide the tools to create the interactable, custom world that you see here! Unity is typically more suited for projects that use AR.

Using C#

C# is the language that Unity uses to code its components. These components are placed on the game objects to provide the functionality for the game.

And Jira

Jira helped our team keep track of project goals and not fall behind. It was a great tool for measuring progress and documenting work!
