VR Computer Lab

A faithful recreation of the lab at 1695 BBB before recent renovations... mostly.

About the Project

The goal of this project was to recreate a physical space based on photo and video footage. Upon creation of the computer lab, visual and audio components help immerse the user and make the experience feel more real. We also added interactivity to make it a rich environment for the user.

This whole Unreal Engine project was created by myself and one partner, Ashton Thomas, for our EECS 440 class at the University of Michigan. Although the project was created and finished in just a month, I think the final product is rather entertaining, don't you?

Made in Unreal Engine 5

Unreal Engine is one of two major game engines widely used today. Along with Unity, these game engines provide the tools to create an interactable, custom world like the one I made in this project!

Using Blueprints

Blueprints is Unreal Engine's visual coding language. Although code can also be written with C++, the standard language used in Unreal is this node-based language.

And Jira

Jira helped our team keep track of project goals and not fall behind. It was a great tool for measuring progress and documenting work!
